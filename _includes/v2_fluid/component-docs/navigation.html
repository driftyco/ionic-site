Navigation

Improve this Doc Navigation is how users move between different views in your app. Ionic's navigation follows standard native animation concepts, like [those in iOS](https://developer.apple.com/library/ios/documentation/UserExperience/Conceptual/MobileHIG/Navigation.html). Views are pushed on and popped off of a navigation controller's view stack. _For more information, Check out the [API docs for NavController](../api/components/nav/NavController), the [API docs for NavBar](../api/components/navbar/Navbar/), and the [API docs for Nav](../api/components/nav/Nav)._
