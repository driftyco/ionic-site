Loading

Improve this Doc The Loading component is an overlay that prevents user interaction while indicating activity. By default, it shows a spinner based on the mode. It can also be passed any HTML content to display with the spinner. You can hide or change the spinner to any of our predefined spinners. The loading indicator is styled to display on top of other content even during navigation. Check out the [API docs](../api/components/loading/Loading) for more information.

Basic Usage

Demo Source ```typescript presentLoading() { let loading = Loading.create({ content: "Please wait...", duration: 3000 }); this.nav.present(loading); } ``` Alternatively you can show and hide the loading component based on async calls using the returned `ViewController` object. ```typescript presentLoading() { let loading = Loading.create({ content: "Please wait..." }); this.nav.present(loading); //performing some form of async call, e.g. a http request to a server. backend.someAsyncCall().subscribe(response=>{ loading.dismiss(); }); } ```
