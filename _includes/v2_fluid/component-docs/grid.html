Grid

Improve this Doc Ionic's grid system is based on [flexbox](http://www.w3.org/TR/css3-flexbox/), a CSS feature supported by all devices that Ionic supports. The grid is composed of two units — _rows_ and _columns_. Columns will expand to fill their row, and will resize to fit additional columns. To vertically align the columns, assign one of the following attributes: `baseline`, `center`, `top`, `bottom` or `stretch`. You can also assign specific widths in percentage of the total width using the `width` attribute (e.g. `width-10`, `width-33`, `width-50` and so on), or you can offset the column with the `offset` attribute. ```html
