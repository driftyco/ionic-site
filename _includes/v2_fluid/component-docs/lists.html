Basic Usage (Default)

View Demo Source By default, all lists will be styled with divider lines: ```html {% raw %}{{item.title}}{% endraw %} ```

Basic Usage (No Lines)

View Demo Source Adding the `no-lines` attribute will hide the dividers between list items: ```html {% raw %}{{item.title}}{% endraw %} ```

Inset List

View Demo Source By default, lists have an outside margin, to remove that add the `inset` property to make the list flush with the parent container. ```html {% raw %}{{item.title}}{% endraw %} ```

List Headers

View Demo Source Each list can include a header at the top of the list: ```html Action Terminator II The Empire Strikes Back Blade Runner ```

Icon List

View Demo Source Adding [icons](#icons) to list items is a great way to hint about the contents of each item. The position of the icon can be set using the `item-left` and `item-right` attributes: ```html Herbology ```

Multiline List
