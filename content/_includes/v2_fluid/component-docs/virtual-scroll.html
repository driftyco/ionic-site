Virtual Scroll

Improve this Doc The Virtual Scroll component is one solution Ionic offers for peformantly rendering large amounts of data, and is particularly useful for very long lists. Virtual Scroll is used in place of `*ngFor` for iterating over records in an array, and works by rendering only enough records to fill the viewport. Virtual Scroll then reuses the DOM elements for those records as the user scrolls. This prevents very large arrays from rendering a large number of elements to the DOM, which can result in performance issues. _For complete usage information, check out the [Virtual Scroll API docs](../api/components/virtual-scroll/VirtualScroll)._ Demo Source ```html {{ item }} ```
