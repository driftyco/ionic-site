Infinite Scroll

The Infinite Scroll component is one solution Ionic offers for peformantly rendering large amounts of data, and is particularly useful for very long lists. Infinite Scroll calls a specified function when the user scrolls a set distance from the bottom of the page. This allows you to do things like fetch additional data from an API, then add it to a list to be rendered.
