Gestures

Improve this Doc Basic gestures can be accessed from HTML by binding to `tap`, `press`, `pan`, `swipe`, `rotate`, and `pinch` events.

Basic Usage

Demo Demo Source

```html Tapped: {% raw %}{{tap}}{% endraw %} times ``` ```html Swiped: {% raw %}{{swipe}}{% endraw %} times ``` Example of the .ts code to capture the events: ```html swipeEvent(e) { if (e.direction == 2) { //direction 2 = right to left swipe. } } ```

Swipe directions include: Value - Name

  • DIRECTION_NONE - 1
  • 2 - DIRECTION_LEFT
  • 4 - DIRECTION_RIGHT
  • 8 - DIRECTION_UP
  • 16 - DIRECTION_DOWN
  • 6 - DIRECTION_HORIZONTAL
  • 24 - DIRECTION_VERTICAL
  • 30 - DIRECTION_ALL
